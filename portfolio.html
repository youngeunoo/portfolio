    Portfolio

    포트폴리오 페이지를 기획하고 구현하여 제작한 프로젝트를 소개한다.

    개인 프로젝트 개요

    • 작업 기간 : 23. 12. 27 - 24. 01. 04
    • 기여도 : 100%

    목표

    • 포트폴리오를 통해 프로젝트와 가진 기술을 소개함.
    • 심미적인 디자인을 통해 사용자가 보기 편하도록 함.
    • 전하려는 정보가 한눈에 보기 쉽게 읽힐 수 있도록 함.

    작업 순서

    • 사전 기획
      • 들어갈 내용 정리
    • 디자인 작업
      • 메인 페이지와 프로젝트 상세 페이지 디자인
      • 배경 색상, 구성 색상, 폰트 선정
      • 필요한 아이콘, 이미지 요소 선정
    • 개발 작업
      • html 마크업 작업
      • css 스타일 작업
      • GSAP의 ScrollTrigger 사용하여 skill 영역 효과 구현
      • outro 영역 애니메이션 적용
      • project 영역 swiper 적용
      • 모바일 페이지 작업

    사용 기술 / 도구

    • HTML5
    • CSS3
    • JaveScript
    • GSAP
    • Swiper
    • Figma

    디자인 가이드

    • 키워드
      • 따뜻함
      • 깔끔함
      • 차분함
    • 색상

      • 포인트 색상

      • 배경 색상

    • 폰트

    퍼블리싱

    • GSAP의 ScrollTrigger를 사용하여 skill 영역의 내용들이 차례대로 보여질 수 있도록 하였다.
    • Swiper를 적용하여 project 영역의 내용을 차례대로 확인할 수 있도록 하였다.

    문제 및 해결

    • position으로 이미지 요소들을 배치했는데, 화면 사이즈가 조금만 달라져도 원하는 위치에서 벗어나 버렸다.

      → div로 요소들을 세세하게 나누고 배치하여 위치 배정에 용이하도록 수정하였다.

    • project 영역에서 각 프로젝트에 마우스를 올렸을 때 어떤 효과를 줄지 고민하였다.

      → 어둡게 하기, 돋보기 효과 나타나기 등 여러 가지를 적용해보다가 흑백에서 컬러로 변하게 하는 것으로 선택하였다.
